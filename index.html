Question 18

What are two characteristics of ARP? (Choose two.)

Explanation: When a node encapsulates a data packet into a frame, it needs the destination MAC address. It first determines whether the destination device is on the local network or a remote network. It then checks the ARP table (not the MAC table) to see if there is a pair of IP and MAC addresses for the destination IP address (if the destination host is on the local network) or the default gateway IP address (if the destination host is on a remote network). If the match does not exist, it generates an ARP broadcast to resolve the IP address to the MAC address. Because the destination MAC address is unknown, the ARP request is broadcast with the MAC address FFFF.FFFF.FFFF. The destination device or the default gateway will respond with its MAC address, allowing the sending node to assemble the frame. If no device responds to the ARP request, the originating node will discard the packet because a frame cannot be created.
